# REST API

REST API umožňuje přístup k obsahu prostřednictvím koncových bodů API.
Parametry API lze použít při dotazování na koncové body API k upřesnění výsledků.

API obsahuje tyto hlavní sady koncových bodů:

- auth
- articles
- employees
- events
- documents
- gallery
- images
- products
- orders
- notifications
- pricelist
- pages
- vacancies
- users
