# REST API

The REST API allows accessing the content through API endpoints. 
API parameters can be used when querying API endpoints to refine the results.

API contains these main sets of endpoints:

- auth
- articles
- employees
- events
- documents
- gallery
- images
- products
- orders
- notifications
- pricelist
- pages
- vacancies
- users
