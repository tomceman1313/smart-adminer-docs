# Gallery

Within this section it is possible to manage your gallery images.

The following information can be defined for an image:

- Name
- Description
- Category (multiple)
- Image (required)

## Adding an image

You can add an image either individually or add multiple images at once.

### Adding one image

1. In the New Image section, fill in the input fields as needed
2. Click the green **Insert** button to save the new image

![Hello](/images/gallery-single-form.PNG)

### Adding multiple images at once

1. Click on the blue button **Insert more** (A form popup will appear)
2. Select the appropriate category and upload images
3. Click the **Insert** button

![Hello](/images/gallery-multiple-form.PNG)

## Image editing

1. Find the desired image in the list and click on it (Two buttons will appear, a text and a label)
2. Click the green button on the top left (a pop-up window with a form will appear)
3. Make changes
4. Click the **Save** button

![Hello](/images/gallery-image.PNG)

1. Find the desired image in the list and click on it (Two buttons will appear, a text and a label)
