# Orders

Within this section, it is possible to manage received orders.

The following information can be defined for an article / event:

**Information about order:**
- List of ordered products
- Status (required)
- Type of payment (required)
- Method of delivery (required)
- Order date (required)
- Date of departure
- Date of completion
- Notes

**Billing information**
- First name (required)
- Surname (required)
- Phone (required)
- Email (required)
- Address (required)
- City (required)
- Postcode (required)

**Delivery information:**
- First name
- Surname
- Address
- The city
- Postal code

**Purchasing for a company:**
- Name (required)
- ID number (required)
- TIN

## Order filtering

You can filter based on order number, date of receipt, status, mode of transport and payment method.

1. Click on the order table header (a green container will appear on the right side)
2. Select the items by which you want to Filter orders
3. Click the **Filter** button in the lower left corner of the container
4. Cancel the filter by clicking the **Reset** button in the lower right corner of the container

## Edit order

1. Click on the required order in the table
2. Edit the information in the form as needed
3. Click the green **Save** button

<Callout type="warning" emoji="ðŸ’¡">
  The order can be freely edited, but except in exceptional cases, you should only change the **status** and **send date** and **completion**.
</Callout>

## Deleting an order

1. Click on the required order in the table
2. Click the red button **Delete**
3. Confirm the deletion by clicking the **Confirm** button
