# Dynamically managed pages

V rámci této sekce je možné spravovat dynamický obsah stránek. Může se jednat o kontrolu nad celým tělem stránky nebo pouze nad fotkou či jedním odstavcem.

Každá dynamická sekce má vlastní konfigurační sadu, která určuje, jaké vstupní pole jsou uživateli nabídnuty k vyplnění.
Seznam dostupných vstupních polí:

- Title
- Description
- Image
- Rich text editor
- Text area (allows you to insert only text without any formatting)

## Modifying the dynamic section of the page

1. Select the desired page in the list and click the blue button with the arrow icon pointing to the right (available dynamic sections within the given page will be displayed)
2. Select the desired dynamic section you want to edit and click the green button with the ✏ icon (you will be redirected to the editing page)

![Hello](/images/pages.PNG)

3. Edit the information
4. Click the green **Save** button

# Other operations
 
Currently, SmartAdminer only supports the editing of dynamic parts by the user, as other operations are intended exclusively for developers who create a presentation page.
