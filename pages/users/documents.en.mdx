# Documents

Documents can be managed within this section.

The following information can be defined for a document:

- Title (required)
- Description
- Category (required)
- Date of addition (if not filled in, it is set to the current day at the time of addition)
- Preview image
- File (required)

## Adding a document

You can add a document either individually or add several at once.

### Adding one document

1. In the New Image section, fill in the input fields as needed
2. To save a new document, click the green button **Insert**

![Hello](/images/documents-single-form.PNG)

### Adding multiple documents at once

1. Click on the blue button **Insert more** (A form popup will appear)
2. Select the appropriate category and upload images
3. Click the **Insert** button

![Hello](/images/documents-multiple-form.PNG)

## Document modification

1. Find the desired document in the list and click it (Three buttons will appear)
2. Click the green button in the middle (a form popup will appear)
3. Make changes
4. Click the **Save** button

![Hello](/images/documents-document.PNG)
